Side Projects

My side projects showcase my problem-solving skills, with accomplishments such as creating both heuristic and backtracking solutions. The Sudoku solver program uses a backtracking algorithm in which the computer iterates through each open space, placing the first available number 1 through 9. As soon as the code interacts with an error, when the number doesn’t fit the requirements, the code moves back one space and chooses a different available number. The algorithm continues until it reaches a solution or exhausts all possibilities.

The 2048 generator uses a variety of libraries, the board starting off as an array of 0’s. Once the game is in motion, the game inputs two random numbers, either a 2 or a 4, to begin the game. The player is able to move in any direction; left, right, up, or down. After every move, the board adds another tile at random until the game board is full or the player reaches 2048. The solver uses a heuristic algorithm to find the best route to win the game; to reach 2938. The game solver measures the situation of the board, and considers what the board would be like if it moved left, right, up or down. For each possible move, it considers where the next random could be put and displays a list of possible outcomes. IT finds the heuristic value, in comparison to a set ‘ideal’ board, for each situation of each move, and averages them out to create a heuristic value for every move. THe board decides to play the move with the highest heuristic value until the board is full or reaches 2048. I am currently working on the Traveling Salesman Problem.
